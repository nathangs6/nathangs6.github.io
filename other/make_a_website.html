Making a UofT Website

If you're a student at UofT you get free web hosting through UofT! This website is an example of that. The following is how you do so:

  1. Go here and read the FAQ and conditions and use.
  2. To request the web space you'll be sent to this website (but go through the utorweb website, in the "request web space" section).
  3. Go to "change options" and login using your utorid.
  4. There should be a subsection on that page for "Individual Web Space" where you can request web space.
  5. Within a day you should have a website ready for use!
To modify your website you'll need to do the following:
  1. Learn some HTML (and optionally CSS). This shouldn't take too long to learn so don't be intimidated by it! Any tutorial on the internet should be sufficient for basic use. If you want to use the ones I used, go to my "Certificates" page.
  2. A website works like a file system, just accessible from anywhere. So to change your website you need to change the files in your website's "folder". To do so, do the following:
    1. First, go to the same page that you used to request your web space. At the top of the page, you should see a section called "Address for saving (Uploading)".
    2. Copy and paste this address into your file explorer and log in.
    3. Now you can add, remove, and modify your website's files!
    4. Note: The file called index.html will be what a visitor will be brought to when they go to your website's url, so you should always have an index.html file. For example, if you type http://individual.utoronto.ca/nathangs/ into your search bar, you will actually be brought to http://individual.utoronto.ca/nathangs/index.html.
    5. You may need to clear your web cache to see the changes.

Tip: In most browsers you can view the source code for any webpage you are on. On Windows 10, Firefox, you can do this by right clicking and selecting "View Page Source". You can use this as inspiration! You can also use "Inspect Element" to get a more detailed look.
